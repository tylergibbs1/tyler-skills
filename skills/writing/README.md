# Writing

Copy skills that keep product writing accurate and human-sounding. All model-invoked.

- **[grounded-product-copy](./grounded-product-copy/SKILL.md)**: Only claim what the product actually does. Reads a `PRODUCT.md` ground-truth file before writing, so copy never invents features, testimonials, or scope.
- **[human-sounding-copy](./human-sounding-copy/SKILL.md)**: Strip the tells that make copy read as AI-generated or look cheap: no em dashes, no jargon, no all-caps eyebrows, minimal icons.
