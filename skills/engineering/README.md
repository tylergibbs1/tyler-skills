# Engineering

Discipline skills that pre-empt the corrections agents get most often on real code work. All model-invoked: the model reaches for them at the right moment, and you can invoke them by name too.

- **[closing-the-loop](./closing-the-loop/SKILL.md)** — Verify an action actually succeeded before reporting it done. A completion claim must cite an observed result, never the fact that you launched it.
- **[scope-discipline](./scope-discipline/SKILL.md)** — Build exactly what was asked and nothing more. No unrequested features, migrations, caps, or PR ceremony.
- **[read-before-acting](./read-before-acting/SKILL.md)** — Read the real code, render, or data before answering or editing. Pre-empts "just look at the code".
- **[safe-reverts](./safe-reverts/SKILL.md)** — Treat reverts and destructive git operations as high blast radius. Assume parallel edits, confirm before destroying uncommitted work.
